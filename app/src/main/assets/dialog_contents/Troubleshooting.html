If the cursor doesn't move after activating Air Mouse, first make sure that:

  • You removed the S Pen from the phone and waited for it to connect before launching the mouse feature
  • You're using a compatible S Pen that supports Air Actions
  • The S Pen battery is charged
  • You're using a compatible device:
    • Galaxy Note10
    • S20 Ultra - S24 Ultra
    • Galaxy Tabs with S Pen remote support

Please Note: The S25 Ultra no longer supports Bluetooth in the S Pen.

If the mouse feature used to work on this device but suddenly stopped, try the following:

  • Go to Accessibility settings and toggle PenMouse S off and back on
  • Restart your device
