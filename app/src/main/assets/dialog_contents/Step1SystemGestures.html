By default, the S Pen has several built-in system functions that allow it to act as a remote control for your phone. For example, holding down the button on the stylus typically launches the camera.

Normally, system actions of the S Pen are not triggered in apps that actively use the stylus (e.g., in games like Bird Hunt). However, when using the S Pen as an Air Mouse, the pen does not recognize that it's inside an app, meaning all of its default system functions, gestures, and shortcuts remain active.

To ensure the best performance of the Air Mouse mode, we recommend disabling the following S Pen features in the settings:

  • Pointer command icon display
  • Long press on the stylus button
  • All available gesture actions:
    • Back
    • Home
    • Recent apps
    • etc.
